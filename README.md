# 🏏 IPL Match Win Probability Predictor

A complete end-to-end Machine Learning web application that predicts the win probability of an IPL team during a run chase — in real time.

---

## 📸 Features

- **Live Win Probability** — predicts batting & bowling team win chance
- **Visual Output** — animated progress bars + Plotly pie chart
- **Match Summary** — intelligent contextual analysis text
- **Modern UI** — cricket-themed Streamlit sidebar layout

## 🚀 Quick Start (Local)

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/IPL-Win-Predictor.git
cd IPL-Win-Predictor
```

### 2. Create a virtual environment

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add the Kaggle dataset

Download the IPL dataset from Kaggle and place these two files inside the `data/` folder:

- `data/matches.csv`
- `data/deliveries.csv`

### 5. Run the Jupyter Notebook to train the model

Open and run all cells in `IPL_Win_Predictor.ipynb`.  
This will generate `pipe.pkl` in the project root.

```bash
jupyter notebook IPL_Win_Predictor.ipynb
```

### 6. Launch the Streamlit app

```bash
streamlit run app.py
```

Open your browser at: `http://localhost:8501`

## 📊 How It Works

### Training (Notebook)

1. Load `matches.csv` and `deliveries.csv`
2. Filter to **2nd innings** only (run chase data)
3. Engineer features:
   - `runs_left`, `balls_left`, `wickets_left`
   - `crr` (current run rate), `rrr` (required run rate)
   - `target`, `result` (win = 1, loss = 0)
4. Train a **Logistic Regression** model inside a `sklearn Pipeline`
5. Save the pipeline as `pipe.pkl` using `pickle`

### Prediction (Streamlit App)

1. User fills inputs in the sidebar
2. App computes derived features on-the-fly
3. Loads `pipe.pkl` and calls `pipe.predict_proba()`
4. Displays win/loss probabilities with charts

## 🤖 ML Model Details

| Property | Value |
|---|---|
| Algorithm | Logistic Regression |
| Framework | scikit-learn Pipeline |
| Encoder | OneHotEncoder (for team names) |
| Features | batting_team, bowling_team, runs_left, balls_left, wickets_left, target, crr, rrr |
| Target | result (1 = win, 0 = loss) |
| Output | Probability (predict_proba) |
